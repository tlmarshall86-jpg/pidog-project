# Buddy the Robot Dog 🐶

## About
Buddy is a robotic dog built using the SunFounder PiDog kit and Raspberry Pi 4. This project combines hardware, Python programming, and AI features.

## Features
- Pushups
- Tail movement
- Object detection
- Face tracking
- Voice interaction
- Ball tracking

## What I Learned
- Robotics assembly and calibration
- Python programming for hardware
- Troubleshooting real-world issues
- Confidence in problem-solving
